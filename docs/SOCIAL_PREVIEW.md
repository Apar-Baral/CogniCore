# GitHub social preview image

The repository includes a ready-made Open Graph banner:

**[docs/assets/og-banner.png](assets/og-banner.png)** (1200×630)

## Set on GitHub

1. Open **https://github.com/Apar-Baral/CogniCore**
2. **Settings** → **General** → **Social preview**
3. **Upload an image** → select `docs/assets/og-banner.png` from this repo (download or clone first)
4. Save

Link previews on Twitter, Discord, Slack, etc. will use this image.

## README hero

The main [README.md](../README.md) displays the same banner at the top for visitors on github.com.
